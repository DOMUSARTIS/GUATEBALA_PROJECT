# PROTOCOLO SOBERANO - MVP

Vertical slice web del juego de decisiones de crisis.

## Incluye

- Direccion artistica "Estado de Emergencia" (UI sobria de centro de mando).
- Loop principal: incidente -> 3 opciones -> impacto en metricas.
- Deudas diferidas: algunas decisiones explotan 1-2 turnos despues.
- Debrief final con perfil de liderazgo.

## Como correr

1. Abre `index.html` en navegador.
2. Presiona `[ INICIAR PROTOCOLO ]`.
3. Toma 6 decisiones y revisa el resultado final.

## Metricas

- Control
- Legitimidad
- Confianza
- Costo Humano
- Riesgo Futuro
